Mostrar lista de PV (Physical volumen):
# sudo pvs

Crear grupo LV para luego formatear:
# sudo lvcreate -L +XG -n nombreDelLV nombredelVG

Ej: sudo lvcreate -L +6,5G -n ParteGrande DiscoGrande

Eliminar grupo LV:
# sudo lvremove nombreDelLV nombreDelVG

Mostrar los LV, la capacidad y donde están montadas:
# df -h

Crear usuario correctamente:
# useradd nombreUsuario -p contraseña -s /bin/bash -m -g nombreGrupoYaCreado

Crear un grupo para usuarios:
# sudo groupadd nombreGrupo

Cambiar permisos a un directorio:
# sudo chown -R nuevoUserDueño:grupoDelUser /destino/

Cambiar contraseña a usuario:
# sudo passwd nombreUsuario

Desmontar un PV:
# sudo udisksctl unmount -b 

Como eliminar /home/ de otros usuarios:
#sudo userdel -f nombreUsuario (elimina usuario y su home)
--Si se eliminó el usuario antes de eliminar el home usar en modo root # rm -r /home/nombreUsuario/
